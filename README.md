## 🔍 Frequently asked questions

Do I need to copy my game folder?
No. The tool works directly with your game files. We suggest you create a backup of your save files before starting, however, just in case you want to revert your progress.

Will this work with online play?
No. Modding files triggers anti-cheat software. Only use these tools for offline career modes. Playing online with modified files may lead to a temporary ban from the game servers.

My changes do not appear in the game. What do I do?
Ensure the mod folder path is correct in the Settings menu. Check that you clicked the Save button after making changes. Refresh the tool by closing it and reopening it.

The tool shows an error message.
Check your internet connection first. Ensure that your antivirus software does not block the application. You can add an exception for the FC26-Mod-Editor-X folder in your antivirus settings.
